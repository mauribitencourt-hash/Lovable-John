## 🚀 Como Começar (3 passos)

### 1. Recarregar Extensão (30 segundos)
```
1. Abra chrome://extensions
2. Procure "Painel Lovable"
3. Clique no ícone de recarregar
```

### 2. Testar Device ID (1 minuto)
```
1. Clique no ícone da extensão
2. Veja o painel VERDE no lado direito
3. Clique em "📋 COPIAR"
4. Veja "✅ Copiado!" aparecer
```

### 3. (Opcional) Fixar para Múltiplos PCs (2 minutos)
```
1. Copie o Device ID
2. Abra deviceConfig.js
3. Linha 21: FIXED_DEVICE_ID = 'ID_AQUI';
4. Recarregue extensão
5. Todos os PCs com este arquivo terão o mesmo ID
```
