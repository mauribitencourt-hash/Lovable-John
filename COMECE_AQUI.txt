  PRIMEIRA VEZ:
    1. Extensão carrega → systemDeviceIdIntegration.js
    2. Verifica storage → Vazio? Gera novo ID!
    3. Native Messaging → Conecta ao PowerShell
    4. PowerShell coleta → SID, Hostname, MachineGuid, etc
    5. Sistema calcula → SHA-256 hash (64 caracteres hex)
    6. Armazena → chrome.storage.local (permanente)
    7. Retorna → Device ID único

  PRÓXIMAS VEZES:
    1. Extensão carrega
    2. Verifica storage → Tem ID! Retorna imediatamente ✅
    3. Sem comunicar com PowerShell (rápido!)

  OUTRO NAVEGADOR (mesmo PC):
    1. Firefox abre extensão
    2. Verifica seu storage → Vazio
    3. Native Messaging → Mesmo Windows = Mesmo hash!
    4. Retorna → MESMO Device ID do Chrome ✅
